# MIB Beacon 0.0.1-beta.1

Initial public beta of the cross-platform MIB/SNMP workbench.

## Included

- Local MIB import, diagnostics, catalog browsing, translation, and table semantics.
- Encrypted agent profiles; Get/GetNext/GetBulk/Set/Walk/Table operations and exports.
- Trap receive, search, rules, replay, and v1/v2c/v3 send/inform workflows.
- Consent-gated recursive online/custom-source resolution with private cache.
- Polling graphs, watches, discovery, compare, port monitoring, and reachability tools.
- Responsive phone/tablet/desktop shells, semantic themes, keyboard controls, and accessibility labels.
- Linux, Windows, macOS, Android, unsigned iOS, and Flatpak release automation.

## Important limitations

- This beta is not security-audited; read the README disclaimer before use.
- Mobile trap reception is foreground-only. Expo Go is unsupported.
- Desktop and mobile signing depend on release credentials; unsigned-platform warnings apply.
- Automatic network resolution and automatic desktop update checks start disabled.
- Platform/device validation results are recorded in `docs/RELEASE-CHECKLIST.md`.

## Upgrade notes

This is the first beta. Back up private application data before replacing it.
Verify downloaded artifacts against `SHA256SUMS`.
