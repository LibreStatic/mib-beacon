# MIB Beacon 0.1.0-beta.1

First feature beta release of the cross-platform MIB/SNMP workbench.

## Highlights

- Hardened LAN server credential-key startup and Docker Compose bootstrap behavior.
- Expanded Tools workflows with validated browser E2E coverage.
- Signed Android APK/AAB builds use the configured release key.
- Linux packages, Flatpak, signed Android APK/AAB, unsigned iOS IPA, and clearly labeled unsigned Windows NSIS and macOS DMG artifacts are built by the tag workflow.

## Important limitations

- This beta is not security-audited; read the README disclaimer before use.
- Mobile trap reception is foreground-only. Expo Go is unsupported.
- The Windows and macOS artifacts in this beta are intentionally unsigned; operating-system warnings are expected.
- Automatic network resolution and automatic desktop update checks start disabled.
- Paid signed Windows/macOS release paths remain opt-in and require their documented credentials.

## Upgrade notes

Back up private application data before replacing an existing beta. Verify downloaded artifacts against `SHA256SUMS`.
